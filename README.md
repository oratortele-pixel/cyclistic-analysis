# Cyclistic Bike-Share Analysis

## Overview
Analysis of 800,000+ bike trips to understand how casual riders and annual 
members use Cyclistic bikes differently.

## Key Findings
- Casual riders take 3x longer trips than members (36 min vs 11 min)
- Members ride consistently on weekdays — classic commuter behaviour
- Casual riders peak on weekends — leisure behaviour

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

## Data Source
Divvy trip data (2019 Q1 and 2020 Q1) provided by Motivate International Inc.
